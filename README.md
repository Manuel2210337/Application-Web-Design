## Que es el MarkDown?
es un lenguaje de programacion que fue creado por John Gruber en 2004 el proposito de este lenguaje es permitir a los usuarios escribir documentos en un formato de texto plano que sea fácil de leer y escribir, y que pueda convertirse fácilmente en HTML válido esto fue diseñado inicialmente para simplificar la sintaxis de HTML, especialmente en documentos como archivos README.
